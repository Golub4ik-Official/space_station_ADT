- ✅ — пройдено
- ❌ — не пройдено

## 1. Базовая функциональность

### 1.1 Спавн борга

- [ ] 🔲 **Спавн через конструкцию**: собрать борга через конструкцию (граф Cyborg)
- [ ] 🔲 **Спавн через админку**: `spawn ADTBorgEngineering` (или другой тип)
- [ ] 🔲 **Инициализация**: при спавне создаются 6 частей (Armour, Actuator, Radio, BinaryCommunication, Camera, DiagnosisUnit)
- [ ] 🔲 **Мозг**: борг появляется с мозгом (MMI/posibrain)
- [ ] 🔲 **Батарея**: борг появляется с заряженной батареей
- [ ] 🔲 **Законы**: по умолчанию Crewsimov

### 1.2 Взаимодействие

- [ ] 🔲 **Верб Toggle Panel**: открыть/закрыть техпанель
- [ ] 🔲 **Верб Cycle Slot**: переключение между 3 слотами
- [ ] 🔲 **Верб State Laws**: показать текущие законы
- [ ] 🔲 **Поглаживание**: interaction popup при клике по боргу (разные сообщения для разных типов)

### 1.3 Инструменты

- [ ] 🔲 **Отвёртка**: при открытой панели переключает блокировку
- [ ] 🔲 **Монтировка**: с открытой панелью отрывает случайную часть тела
- [ ] 🔲 **Сварка**: чинит все повреждённые части
- [ ] 🔲 **Мультитул**: проверка проводов (с открытой панелью)

## 2. Система питания

### 2.1 Батарея

- [ ] 🔲 **Начальный заряд**: батарея заряжена при спавне
- [ ] 🔲 **Разряд**: с течением времени заряд уменьшается (drainPerSecond: 10)
- [ ] 🔲 **Низкий заряд**: при заряде < 10% появляется предупреждение
- [ ] 🔲 **Полный разряд**: при 0% борг отключается (становится неактивным)
- [ ] 🔲 **Зарядка**: в станции зарядки (borg_charger) батарея восстанавливается

### 2.2 Элемент питания

- [ ] 🔲 **Извлечение**: извлечение батареи через слот cell_slot
- [ ] 🔲 **Установка**: установка заряженной батареи восстанавливает питание
- [ ] 🔲 **Потребление**: использование способностей тратит энергию

## 3. Система частей и урон

### 3.1 Нанесение урона

- [ ] 🔲 **Распределение**: blunt/slash/pierce → brute; heat/shock → burn
- [ ] 🔲 **Броня**: первые удары поглощаются Armour частью (100 HP)
- [ ] 🔲 **Случайные части**: после пробития брони урон идёт в случайные части
- [ ] 🔲 **Смерть**: при превышении макс. здоровья (100) борг погибает

### 3.2 Ремонт

- [ ] 🔲 **Сварка**: чинит все части сразу
- [ ] 🔲 **Удаление части**: монтировка с открытой панелью отрывает часть
- [ ] 🔲 **Урон части**: при достижении макс. HP части она ломается (broken)

### 3.3 Смерть и гиб

- [ ] 🔲 **Смерть**: при 100+ урона борг падает (становится неактивным)
- [ ] 🔲 **Гиб**: при достижении порога Destructible (100 урона) — гиб
- [ ] 🔲 **Мозг**: при гибе мозг извлекается
- [ ] 🔲 **Перезагрузка**: ADTBorgUpgradeRestart запускает перезагрузку (15с)

## 16. Граничные случаи

- [ ] 🔲 **Без мозга**: борг без мозга неактивен
- [ ] 🔲 **Без батареи**: борг без батареи не работает
- [ ] 🔲 **Поломка всех частей**: при поломке всех частей борг умирает
- [ ] 🔲 **Множественные EMAG**: повторный EMAG не накладывается
- [ ] 🔲 **Апгрейд без модуля**: апгрейд с `requiredModule` не работает без соответствующего модуля
- [ ] 🔲 **Смена модуля при полных слотах**: старый модуль извлекается перед установкой нового
- [ ] 🔲 **Провода при перезагрузке**: состояние проводов сохраняется
- [ ] 🔲 **Законы при перезагрузке**: сбрасываются к Crewsimov
